# Modeling-Astrocyte-Mediated-Neuroinflammation-via-RAGE-NF--B-Signaling-Python-Model
This is a computational time course that uses normal differential equation to generate relevant information on NF-kb, Caspase-3, and IL6 in a neurological cell under the effect of Alzheimer's Disease. 

NOTE: This model was intented to run on Python Version 3.12, NumPy Version 1.24, and SciPy Version 1.11. The trapz function of NumPy is deprecated as of NumPy Version 2.0.
